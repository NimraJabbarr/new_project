🎯 WHAT IS RAILWAY?
════════════════════════════════════════════════════════════════════════════════

• No 500 errors on admin
• Automatic deploys from GitHub
• Free tier available ($5-10/month for production)
• PostgreSQL included

• Much simpler than PythonAnywhere


⚡ DEPLOYMENT IN 5 MINUTES
════════════════════════════════════════════════════════════════════════════════

Step 1: Create Railway Account
  1. Go to https://railway.app
  2. Click "Login" → "Sign up with GitHub" (or email)
  3. Authorize Railway to access your GitHub

Step 2: Create New Project
  1. Click "New Project"
  2. Select "Deploy from GitHub repo"
  3. Select your School CRM repository
  4. Click "Deploy"

Step 3: Add PostgreSQL Database
  1. Click "+ New" → Select "PostgreSQL"
  2. Click "Add"
  3. Railway auto-creates database

Step 4: Add Environment Variables
  1. Click "Variables" tab
  2. Add from your .env file:
     
     SECRET_KEY=django-insecure-!1h(7w=t1hg_2p6_x2xy7t#9b-w0_+8&r2004=ga#=l(mhr_er
     DEBUG=False
     DB_ENGINE=django.db.backends.postgresql
     ALLOWED_HOSTS=your-railway-domain.up.railway.app
     JWT_ALGORITHM=HS256
     JWT_EXPIRATION_HOURS=24

Step 5: Railway Auto-Fills Database Variables
  Railway automatically provides:
  • DB_HOST (from PostgreSQL plugin)
  • DB_NAME
  • DB_USER  
  • DB_PASSWORD
  • DB_PORT

Step 6: Deploy!
  1. Click "Deploy" button
  2. Watch logs (should see "Application is running")
  3. Click "Generate Domain"
  4. Your app is LIVE! ✅

Step 7: Run Migrations
  1. Click "Connect" → "Bash console"
  2. Run: python manage.py migrate --noinput
  3. Run: python manage.py createsuperuser (or use existing)
  4. Done!

🐛 TROUBLESHOOTING RAILWAY
════════════════════════════════════════════════════════════════════════════════

If you get 500 error on Railway:
  1. Click "Logs" tab in Railway dashboard
  2. Scroll to bottom to see error message
  3. Common fixes:
     • Make sure all environment variables are set
     • DATABASE_URL should auto-generate from PostgreSQL plugin
     • Check DEBUG=False is set
     • Run migrations: python manage.py migrate --noinput

Admin still showing 500?
  1. Run collectstatic:
     python manage.py collectstatic --noinput --clear
  2. Click "Redeploy"
  3. Wait 30 seconds and refresh
