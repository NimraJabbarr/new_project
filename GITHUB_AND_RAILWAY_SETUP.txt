🐛 IF SOMETHING GOES WRONG
════════════════════════════════════════════════════════════════════════════════

"fatal: remote origin already exists"?
  Run: git remote remove origin
  Then run Command 1 again

"Authentication failed"?
  Use Personal Access Token instead of password
  (See instructions in Command 2)

"Repository not found"?
  Make sure your GitHub username is correct
  Check you created the repository

Railway shows error?
  Click "Logs" tab and scroll to bottom
  Look for error message
  Common fix: Run migrations in Bash console
